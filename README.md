# My website

GitHub Pages based on [Hugo Academic CV Theme](https://github.com/HugoBlox/theme-academic-cv).

## Run

- Install `hugo` to build page

### MacOS

```sh
brew install hugo
```

### Run server locally

```sh
make server
```

### Update hugo modules

```sh
make update
```
